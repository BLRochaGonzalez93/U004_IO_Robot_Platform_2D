# IO Robot Platform 2D Visual Pack

[English](README.en.md) | [Español](README.md)

Planned images for the **IO Robot Platform 2D** project.

This directory is prepared to include the project's visual assets in original and web-optimized versions.

## Planned contents

### original-png/

PNG versions separated from the mockup or original artwork.

- `iorobot-logo.png`
- `iorobot-cover.png`
- `iorobot-banner.png`
- `iorobot-thumbnail-01-mode-switching.png`
- `iorobot-thumbnail-02-platforming.png`
- `iorobot-thumbnail-03-door-button-puzzle.png`
- `iorobot-thumbnail-04-combat-mode.png`

### webp/

The same images converted to `.webp` for web usage.

- `iorobot-logo.webp`
- `iorobot-cover.webp`
- `iorobot-banner.webp`
- `iorobot-thumbnail-01-mode-switching.webp`
- `iorobot-thumbnail-02-platforming.webp`
- `iorobot-thumbnail-03-door-button-puzzle.webp`
- `iorobot-thumbnail-04-combat-mode.webp`

### _source/

Original complete composite image, in PNG and WEBP.

- `iorobot-generated-composite.png`
- `iorobot-generated-composite.webp`

## Recommended usage

- `iorobot-logo.webp` for project identity.
- `iorobot-cover.webp` for Open Graph, cover and social sharing.
- `iorobot-banner.webp` for the portfolio website.
- Thumbnails for cards, documentation, README files or technical previews.
