# IO Robot Platform 2D Visual Pack

[English](README.en.md) | [Español](README.md)

Imágenes previstas para el proyecto **IO Robot Platform 2D**.

Este directorio está preparado para incluir los recursos visuales del proyecto en versiones originales y optimizadas para web.

## Contenido previsto

### original-png/

Versiones PNG separadas desde el mockup o artes originales.

- `iorobot-logo.png`
- `iorobot-cover.png`
- `iorobot-banner.png`
- `iorobot-thumbnail-01-mode-switching.png`
- `iorobot-thumbnail-02-platforming.png`
- `iorobot-thumbnail-03-door-button-puzzle.png`
- `iorobot-thumbnail-04-combat-mode.png`

### webp/

Las mismas imágenes convertidas a `.webp` para uso en web.

- `iorobot-logo.webp`
- `iorobot-cover.webp`
- `iorobot-banner.webp`
- `iorobot-thumbnail-01-mode-switching.webp`
- `iorobot-thumbnail-02-platforming.webp`
- `iorobot-thumbnail-03-door-button-puzzle.webp`
- `iorobot-thumbnail-04-combat-mode.webp`

### _source/

Imagen compuesta original completa, en PNG y WEBP.

- `iorobot-generated-composite.png`
- `iorobot-generated-composite.webp`

## Uso recomendado

- `iorobot-logo.webp` para identidad del proyecto.
- `iorobot-cover.webp` para Open Graph, portada y redes.
- `iorobot-banner.webp` para la web portfolio.
- Las miniaturas para cards, documentación, README o previews técnicas.
