## Contents

- Side-view 2D platformer starring a humanoid robot.
- One functional level with enemies, traps and moving platforms.
- Switching between combat mode and mobility mode.
- Combat mode with projectile shooting.
- Mobility mode with higher speed and double jump.
- Bullet limit and recharge system.
- Timer, lives, score, victory and defeat systems.
- Main menu, pause, sound, music and particles.

## How to run or install

1. Open the GitHub Releases page linked above.
2. Download the build file available in the **Assets** section.
3. Extract the `.zip` into a local folder.
4. Run the game's `.exe` file.
5. Keep the `.exe` inside the extracted folder, together with the internal files generated by Unity.
